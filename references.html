REFERENCES

Command Description Example
echo Echo the STRING(s) to the standard output Type: echo hello world;
Display: hello world
date Display the current time in the given FORMAT, or set the system date Type: date;
Display: Mon 29 Apr 09:11:44 PDT 2019
arch Print machine architecture Type: arch;
Displayl armv71
uname Print certain system information Type: uname -m;
Display: armv17
who Print information about users who are currently logged in Type: who;
Display: sp1a pts/0 2019-04-29 09:00 (172.19.217.139)
top Provides a dynamic real-time view of a running system Type: top;
Display: *displays a large data table of the running system
w Displays information about the users currently on the machine, and their processes Type: w;
Display: 09:21:17 up 1:34, 2 users, load average: 0.07, 0.21, 0.19 USER TTY FROM LOGIN@ IDLE JCPU PCPU WHAT sp1a pts/0
cal, ncal Displays a cimple calendar in traditional format Type: cal 2000;
Display: *year 2000 calendar
sleep Pauses the system for NUMBER seconds Type: sleep;
Display: *nothing*, pauses for 3 seconds
history >Helps keep track of every line in the user's history Type: history;
Display: *displays every command the user has used*
